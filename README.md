# Energy Core - Will Hunt
Roll n Collect Game - Energy Core


- Started 31/03/2026
- Made on Unity

This Roll & Collect game ‘Energy Core’ is a simple 3D Unity game where the player controls a small energy sphere inside a damaged facility. The aim of the game is to move through the environment and collect scattered energy cubes. Once all cubes are collected, the exit door unlocks and the player can escape to complete the level. The game is designed to be small in scope and achievable for a beginner in Unity while still feeling like a complete playable experience.
